# Changelog

이 프로젝트의 모든 주요 변경 사항을 기록합니다.

## [0.6.0] - 2026-07-21

### Phase 3: 학습 모드

- 빈칸 채우기 (Cloze Test): 난이도 3단계, 한글 힌트, 정답 비교
- 받아쓰기 (Dictation): 문장 재생 → 타이핑 → diff 비교 하이라이트
- 학습 모드 선택 페이지 (영상별 빈칸/받아쓰기 선택)
- 영상 상세 페이지에서 학습 모드 바로가기
- 진행률 바 + 완료 시 정답률 통계

## [0.5.0] - 2026-07-21

### Phase 2 완료: 단어·숙어 자동 추출

- 자막 기반 단어 자동 추출 (불용어 필터, 3글자 이상)
- 뉴스 빈출 숙어/구동사 170개 자동 매칭
- 사전 조회 + 한글 번역 자동 처리 (배치, 속도 제한 대응)
- 영상당 1회 추출, 기존 단어장과 중복 제거
- 단어장: 전체/숙어/복습 대상 필터
- 사전 팝업: 영어 정의 + 한글 번역 + 발음 재생
- SM-2 알고리즘 플래시카드 (간격 반복)

## [0.4.0] - 2026-07-21

### Phase 2: 단어·숙어 학습

- 자막 내 단어 클릭 시 Free Dictionary API 사전 팝업
- 발음 재생 버튼 (Audio API)
- 단어장 저장/삭제, 전체/복습 대상 필터
- SM-2 알고리즘 기반 플래시카드 (간격 반복 학습)
- 채점: 모름(1일) / 어려움 / 보통 / 쉬움 → 다음 복습일 자동 계산

## [0.3.0] - 2026-07-21
