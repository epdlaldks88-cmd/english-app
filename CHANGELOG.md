# Changelog

이 프로젝트의 모든 주요 변경 사항을 기록합니다.

## [1.1.0] - 2026-07-22

### 기사 학습 기능

- 영어 뉴스 기사 등록 (URL 자동 추출 + 직접 입력)
- CNN data-component-name 전용 파싱으로 첫 문단 정확 추출
- 문장 단위 분리 + DeepL 한글 번역
- 단어/숙어 자동 추출 (영상과 동일 로직)
- 단어 클릭 시 사전 팝업
- 네비게이션 기사 탭 + 홈 퀵 액션

### Supabase 전체 동기화

- 영상, 자막, 번역, 단어, 기사 멀티디바이스 동기화
- 설정 페이지: 업로드/다운로드 버튼
- 영상 등록/삭제/SRS 업데이트 시 자동 싱크
- 5분 주기 백그라운드 싱크

### UI/UX 개선

- 단어장 페이지네이션 (30개씩, 1000개 이상 먹통 해결)
- 단어장 기사 소스 필터 추가
- 모바일 하단 탭바 콘텐츠 가림 수정
- 자막 병합: 단어 단위 마침표 감지
- 자막 전체 초기화 후 자동 재추출
- 영상 삭제 시 Supabase 연동 삭제
- 설정 페이지: 로컬/전체 초기화 기능 (2중 확인)

### 배포

- Vercel 배포 + Edge Runtime
- Cloudflare Worker 자막 프록시 시도
- Supabase 자막 공유로 멀티디바이스 해결
